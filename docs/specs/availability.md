# Availability

## Scheduling Model

- 후보 시간대는 방 단위 입력이다.
- 참여자는 가능한 시간을 직접 고르지 않고 불가 시간을 입력한다.
- 시스템은 후보 시간대에서 전체 불가 시간 합집합을 빼서 공통 가능 구간을 계산한다.

## Calculation Responsibilities

- 시간 블록 정규화
- 겹침 금지 또는 병합
- 공통 가능 구간 계산
- 결과 순위화

## Stability Rules

- 계산 결과는 같은 입력이면 재현 가능해야 한다.
- 시간 계산 규칙이 바뀌면 관련 스펙 문서를 함께 갱신한다.
- 외부 추천 시스템 장애와 무관하게 시간 결과는 조회 가능해야 한다.
